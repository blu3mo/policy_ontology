# ベストプラクティス調査メモ

このスターターは、Claude Code / Agent Skills / subagents の公式ガイドで推奨される考え方を踏まえている。

反映しているポイント:
- `CLAUDE.md` に持続的なプロジェクト指示を置く
- Skills は再利用可能な単位に分ける
- Skill 名は英小文字ハイフンを維持する
- `description` を具体的にして自動発火しやすくする
- 大きな探索は subagent に逃がして親コンテキストを軽く保つ
- hooks は決定論的ガードレールとして使う
- 可視化や QA は収集・抽出から分ける
- 具体テーマ専用ではなく、複数テーマを横断する共通観点を上位ルールに置く

このため、日本語運用を前提にしつつも、Skill 名や subagent 名は仕様に合わせて ASCII のままにしてある。
