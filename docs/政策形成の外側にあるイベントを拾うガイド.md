# 政策形成の外側にあるイベントを拾うガイド

このメモは、「国政調査」と言っても永田町・霞が関の公式過程だけを見れば十分ではない、という前提を明文化した補助ガイドです。

## 外側のイベントとは何か

ここでいう外側のイベントとは、法案提出や国会質疑そのものではないが、その後の政治判断・行政対応・制度変更に影響した出来事です。

例:
- 詐欺広告被害の拡大
- SNS 事業者の規約変更
- 重大事故やシステム障害
- 消費者団体の調査公表
- 自治体による先行条例
- 海外規制当局による処分や新ルール
- 裁判所の判断
- 学会や研究者の報告書
- 業界団体の自主基準改定

## どう扱うか

1. 外側のイベントも独立したイベントとして記録する。
2. ただし、国会答弁や法案提出とは別イベントにする。
3. そのイベントが制度過程にどう接続したかを、資料付きでリンクする。
4. 接続が弱い場合は、無理に強い因果を主張しない。

## よくあるリンク例

- 被害実態調査 → 国会質疑で引用
- 海外規制強化 → 国内審議会で参考資料化
- 自治体条例 → 国レベルの制度設計論点化
- 企業の自主対応 → 行政処分回避や追加規制議論
- 裁判所判断 → 通知改正や立法論点化

## 推奨メモ

外部イベントを記録するときは、summary か notes_uncertainty に次の観点を残す。
- 何が起きたか
- 誰が関わったか
- どの制度論点に関係するか
- 日本の国政過程との接点は確認済みか、仮説段階か
